# 🚀 SalesPulse AI

### Customer Relationship & Predictive Sales Analytics Platform

SalesPulse AI is a full-stack CRM platform designed to manage customers, deals, and activities efficiently while providing AI-powered insights for deal outcome prediction and customer churn analysis.

---

## ✨ Features

* Secure JWT Authentication
* Customer Management
* Deal Management
* Activity Tracking
* AI-powered Deal Outcome Prediction
* Customer Churn Prediction
* Sales Insights Generation
* Search and Filtering
* RESTful APIs
* Cloud Deployment

---

## 📌 Modules

### 👥 Customer Management

Manage customer information including:

* Name
* Email
* Phone Number
* Company
* Industry
* Employee Count
* Annual Revenue

Supports Create, Read, Update, Delete and Search operations.

---

### 💰 Deal Management

Track business opportunities using:

* Deal Title
* Deal Value
* Deal Stage
* Probability
* Expected Close Date
* Notes

Supports complete CRUD operations and AI-powered deal outcome prediction.

---

### 📅 Activity Tracking

Manage customer interactions through:

* Phone Calls
* Meetings
* Emails
* Follow-ups

Track activity outcomes and maintain engagement history.

---

### 🤖 AI Features

* Deal Success Prediction
* Customer Churn Prediction
* Sales Insights and Analytics

## ☁️ Deployment

The project is fully deployed on cloud platforms.

### Backend Deployment

* Django REST Framework backend deployed on Railway
* REST APIs hosted and connected with PostgreSQL database

### Frontend Deployment

* React.js frontend deployed on Vercel
* Integrated with Railway backend APIs

### Database

* PostgreSQL hosted on Railway

### Authentication

* JWT-based secure authentication

### Architecture

```text
React Frontend (Vercel)
          │
          ▼
Django REST API (Railway)
          │
          ▼
PostgreSQL Database (Railway)
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/pgspraveen/salescrm.git
```

### Backend Setup

```bash
cd salescrm

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
